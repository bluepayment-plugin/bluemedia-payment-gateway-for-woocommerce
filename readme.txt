== Installation	 ==

Z poziomu panelu administratora Wordpress:


1. Przejdź do Wtyczki -> dodaj nową.

2. Kliknij "Wyślij wtyczkę na serwer".

3. Załaduj plik .zip z wtyczką.

4. Zainstaluj i aktywuj wtyczkę.


= Minimalna konfiguracja =

Z poziomu panelu administratora Wordpress:


1. Przejdź do Woocommerce -> ustawienia -> płatności.

2. W tabeli będzie widoczna metoda "Blue Media". Kliknij w nią

3. Zobaczysz formularz ustawień. Wartośći z pola "tytuł" i "opis" będą widoczne publicznie na stronie płatności.
Zanaczenie pola "Enable whitelabel mode" włącza obsługę indywidualnych kanałów płatności (z przekierowaniem) na stronie płatności (Checkout).
Pola obowiązkowe to: Service ID, Production private Key, oraz Gatewayhost. Te dane znajdziesz w [...].
Pole Sandbox mode uruchamia tryb testowy, który wymaga również wypełnienia pól: Test Service ID oraz Test Private Key.

4. Powróć do tabeli bramek płatności i aktywuj przełącznik w kolumnie "włączono"
